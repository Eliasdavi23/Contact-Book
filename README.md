# 📒 Contact Book

A simple contact management system built with Python, implementing basic CRUD (Create, Read, Update, Delete) operations using dictionaries and a menu-driven interface in the terminal.

---

## 🚀 Features

- ➕ Add contact  
- 🔎 View a specific contact  
- ✏️ Edit existing contact  
- ❌ Delete contact  
- 📋 List all contacts  
- 🚪 Exit program  

---

## 🧠 Concepts used

- Python dictionaries  
- Functions  
- Loops (`while`)  
- Conditionals (`if / elif / else`)  
- User input handling (`input()`)  
- Code organization using functions  
- Menu-driven console application
